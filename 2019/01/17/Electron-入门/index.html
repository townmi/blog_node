什么是Electron

Electron是可以使用HTML，CSS和JavaScript来构建跨平台桌面应用程序的一个开源库。 Electron通过将Chromium和Node.js合并到同一个运行时环境中，并将其打包为Mac，Windows和Linux系统下的应用来实现这一目的。

目录代码说明

  • package.json 这个就是项目的配置文件。

  • main.js 是定义在package.json main字段，electron 执行的时候会查找该字段对于的启动文件，如果没有定义，就会查找一级目录下的index.js文件。总之，这个文件就是项目的启动文件。

  • index.html 在应用里面会开启窗口win = new BrowserWindow({ width: 800, height: 600 }) 窗口都会加载一个页面，通常情况下加载的就是index.html

  • render.js 这个脚本，都一般是窗口页面依赖的脚本

  • npm run start 启动electron应用，会得到下面的视图
